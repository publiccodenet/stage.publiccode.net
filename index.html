Public code is essential
The software running our society is increasingly vital public infrastructure, yet it is often hidden in proprietary code bases, producing privately-owned data, and licensed at ever-increasing cost from outside companies.
Public code enables higher quality, more cost-effective services with less risk and more local control. We exist to help ensure that software-dependent service delivery and governance is more democratically accountable, accessible and sustainable.
