Current projects

We rely on these resources for our everyday work.

The Standard for Public Code

The model for public organizations building their own open source solutions, who want to enable successful future reuse by other organizations.
It includes guidance for policymakers, managers, developers and designers.
See the Standard
Governance game
A simple interactive game to help participants understand what governance means for a codebase, the complexity involved, and key issues to consider.
Also useful as a tool for visualizing existing governance models, and potentially beneficial changes.
Governance game on Github

Our podcast: “Let’s talk about public code”

Our interview series with key people’s interesting perspectives on public code.
Recorded as a live stream and then edited.

Listen to the podcast on Podbean

OSPO Alliance (and OSPO.Zone)
A website to help companies and public institutions discover and understand open source, benefit from it across activities, and grow to host an Open Source Program Office (OSPO).
OSPOs are a key aspect of organizational governance essential in non-technology sectors, including education and public administration.
This is a joint initiative with OW2, Eclipse Foundation and OpenForum Europe.
Visit ospo-alliance.org
Bumperscripter
A script for easily generating variations of a video bumper animation.
We originally created this for our livestreamed podcast, as intro graphics help people joining the livestream know the stream is online, active and about to start.
Bumperscriper on Github
url-check
A script for checking the external links of repositories.
This reduces the need for external link checking for each repository, when many links are the same across repositories. We use this script in publiccodenet-url-check to check the list of external links of several repositories in a single place.
See url-check on Github
Past projects

Though our involvement with these projects has finished, we learned a lot from them. Perhaps you will too?

Smart Cities? Public Code!
(February 2018 - April 2019)
This research project aimed to further develop the concept of public code. It explored how we could understand and create public code, while helping answer questions like which technological and institutional arrangements are needed to shift organizations toward the production of public code.
It was a collaboration between the City of Amsterdam, the Amsterdam University of Applied Sciences and Vurb.Agency.

Read more about “Smart Cities? Public Code!”

publiccode.yml
(2018)
A simple, easy to use, developer-centric, public administration open source project metadata format.
Adopted by the Italian government for all new public code projects in Italy, this standard powers the Italian national open source software catalog.
See publiccode.yml on Github
