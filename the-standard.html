Welcome to the Standard
for Public Code

The Standard for Public Code is a set of criteria that supports public organizations in developing and maintaining software and policy together. Its criteria are aligned with guidelines and best practices of open source software development.

See version 0.8.0 (pre-release)

Published 09 January 2024 on GitHub

Key reasons to use the Standard for Public Code!

1
Make codebases more sustainable, usable, accountable, legible, accessible and open.
2
Increase collaboration on and reuse of open source solutions.
3
Get advice for policy makers, government managers, developers and vendors.
4
It’s useful to all levels of government.
